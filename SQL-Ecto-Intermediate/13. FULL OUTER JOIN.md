# Ecto query for `FULL OUTER JOIN`

#####  `join/5`

`join/5` function is used to do full outer join in `Ecto`. To know more about `join/5` function, visit my [[08. JOIN]] page.

###### `:full` option

Example:

``` Ecto
SqlEcto.Hr.Region |> join(:full, [r], c in SqlEcto.Hr.Country, on: r.region_id == c.region_id) |> select([r, c], [r,c])   |> SqlEcto.Repo.all()
```

Explanation:

So the above query will return all matching and not matching records in both the table.

`|> join(:full, [r], c in SqlEcto.Hr.Country, on: r.region_id == c.region_id) `

- It is accepting `SqlEcto.Hr.Region` schema as first argument. 
- `[r]` is the reference variable for `SqlEcto.Hr.Region` schema.
- `c in SqlEcto.Hr.Country` another schema and reference variable.
- `on: r.region_id == c.region_id)` condition.
